    - setManifest() stores the manifest and subsequent hasEntry/getEntry calls reflect it
    - registerAnimations() calls scene.anims.create() once per animation entry in manifest
    - registerAnimations() uses generateFrameNumbers with correct start/end from manifest frames
    - registerAnimations() defaults frameRate to 8 when not specified in manifest
    - registerAnimations() defaults repeat to -1 when not specified in manifest
    - registerAnimations() skips animation keys that already exist (scene.anims.exists returns true)
    - registerAnimations() with empty manifest calls scene.anims.create() zero times
  </behavior>

    For registerAnimations tests, create a mock scene object:

    const mockScene = {
      anims: {
        exists: (key: string) => false,
        create: vi.fn(),

    Then implement src/sprites/SpriteRegistry.ts with these exact exports:

    Types:
    - AnimationDef: { frames: [number, number]; frameRate?: number; repeat?: number }
    - SpriteManifestEntry: { sheet: string; frameWidth: number; frameHeight: number; animations: Record<string, AnimationDef> }
    - SpriteManifest: Partial<Record<string, SpriteManifestEntry>>

    Functions:
    - setManifest(m: SpriteManifest): void -- stores manifest in module-level variable
    - hasEntry(character: string): boolean -- returns character in _manifest
    - getEntry(character: string): SpriteManifestEntry | undefined -- returns _manifest[character]
    - animKey(character: string, animName: string): string -- returns `${character}-${animName}`
    - registerAnimations(scene: Phaser.Scene, manifest: SpriteManifest): void -- iterates manifest entries, creates animations via scene.anims.create() with generateFrameNumbers. Uses animKey for keys. Defaults: frameRate 8, repeat -1. Skips if scene.anims.exists(key) is true.

    For the registerAnimations function signature, use a minimal interface type for the scene parameter instead of importing Phaser.Scene directly, so the module stays testable:
    ```typescript
    interface AnimScene {
      anims: {
        exists(key: string): boolean;
        create(config: object): void;
        generateFrameNumbers(key: string, config: { start: number; end: number }): object[];
      };
    }
    ```
